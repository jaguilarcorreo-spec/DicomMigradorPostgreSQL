Fuentes autoalojadas para despliegue on-premise sin acceso a internet.

  IBM Plex Sans / IBM Plex Mono  — UI general      (300,400,500,600 / 400,500)
  Space Grotesk / Space Mono     — marca MOVE      (500,600,700 / 400,700)

Subconjuntos: latin y latin-ext para las de UI; solo latin para las de marca,
que únicamente renderizan el wordmark "MOVE" y su tagline (ASCII).

Todas se distribuyen bajo SIL Open Font License 1.1, que permite
redistribución e incrustación. Obtenidas de los paquetes Fontsource (npm),
que reempaquetan los originales de Google Fonts / IBM sin modificarlos.
